BOX-MODEL

Padding

La propiedad padding se utiliza para agregar espacio dentro de un elemento, entre su contenido y su borde. Es decir, el padding es el espacio que hay entre el borde del elemento y su contenido.

El box model (modelo de caja en español) es un concepto fundamental en CSS que describe cómo se representan los elementos HTML en una página web. El modelo de caja se refiere a cómo se dibuja un elemento en la página, incluyendo su tamaño, borde, relleno y margen.
